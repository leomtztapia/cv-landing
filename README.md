# CV Landing - cv.leomtztapia.com

Landing page minimalista para centralizar portafolios de Leonardo Martínez.

## Estructura

- **/** - Landing principal con 3 opciones
- **/cv** - Página del Curriculum Vitae
- **/portafolio-web** - Portafolio de proyectos web

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Despliegue

Este proyecto está configurado para exportación estática (SSG). Puede desplegarse en:
- Vercel
- Netlify
- GitHub Pages
- Cualquier servidor web estático

### En Vercel

```bash
vercel
```

### Con archivo estático del CV

Coloca el archivo `cv-leo-web.html` en la carpeta `public/` para que sea accesible en `/cv-leo-web.html`
